4. 패턴 매칭 알고리즘

   1. 고지식한 알고리즘 _ Brute Force

      : 패턴 내의 문자들을 일일이 비교하는 방식

      시간 복잡도 O(n*n)

      ```python
      p = 'youbin'
      t = 'Hi youbin. How are you today?'
      M = len(p)
      N = len(t)
      
      def BruteForce(p, t):
          i = 0
          j = 0
          while j < M and i < N:
              if t[i] != p[j]:
                  i -= j
                  j = -1
              i = i + 1
              j = j + 1
          if j == M :
              return i - M
          else:
              return -1
      
      print(BruteForce(p,t)) # 3
      ```
