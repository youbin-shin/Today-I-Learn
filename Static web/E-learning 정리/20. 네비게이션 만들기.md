- HTML로 텍스트 네비게이션 구조 작성하기
  1. nav 요소 안에서 작성
  2. 네비게이션 HTML 작성 방법: 목록 형태 → 링크의 목록으로 작성
  3. 목록 작성시 순서는 상관 없음: `<ol>` (순서 있는 목록), `<ul>`(순서 없는 목록)
